## 1 Introduction

ESP32-P4-Core is a high-performance dual-core RISC-V module designed by Viewe, based on the ESP32-P4 chip. It uses ESP32-C6FH4 to expand WIFI6 functionality. It supports a variety of human-machine interaction interfaces, including MIPI-CSI (integrated with Image Signal Processor ISP) and MIPI-DSI interfaces. In addition, it supports common peripherals such as SPI, I2S, I2C, LED PWM, MCPWM, RMT, ADC, UART, and TWAI™. It also supports USB OTG 2.0 HS, Ethernet, and SDIO Host 3.0 for high-speed connections, with its peripherals optimized for low-power applications. The chip integrates a digital signature peripheral and a dedicated key management unit to ensure its security. ESP32-P4-Core is specially designed for high-performance and high-security applications, supporting ultra-large on-chip memory and featuring powerful image and voice processing capabilities. It fully meets the higher requirements of embedded applications in terms of human-machine interface support, edge computing capabilities, and IO connection characteristics.

## 2 Functional Features
### 2.1 CPU
•  32-bit RISC-V dual-core processor up to 400 MHz for HP system 

•  32-bit RISC-V single-core processor up to 40 MHz for LP system

•  Equipped with an ESP32-C6 WIFI/BT coprocessor, it expands functions such as WIFI 6 and Bluetooth 5 through SDIO

### 2.2 Memory
•  128 KB HP ROM

•  16 KB LP ROM

•  768 KB HP L2MEM

•  32 KB LP SRAM

•  8 KB system SPM (Scratchpad Memory)

•  32 MB PSRAM is stacked and encapsulated inside, and a 16MB Nor Flash is connected via the QSPI interface.

### 2.3 Applications
With low power consumption, ESP32-P4 is an ideal choice for IoT devices in the following areas:

•  Smart Home

•  Industrial Automation

•  Health Care

•  Consumer Electronics

•  Smart Agriculture

•  Retail Self-Service Terminals (POS, Vending Machines)

•  Service Robot

•  Multimedia Player

•  Cameras for Video Streaming

•  High-Speed USB Host and Device

•  Smart Voice Interaction Terminal

•  Edge Vision AI Processor

•  HMI Control Pane
